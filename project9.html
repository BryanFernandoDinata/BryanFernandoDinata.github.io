About This Project


This project one of my earliest game Project. I did this project when I was learning about Game Development using Unity Game Engine. This is a 2D Action Sidescroller Game where the player need to dodge obstacles, collect coins, and eliminate enemies in order to win the game. The other reason why I did this project is because I'm bored.


Doing this project was really fun. This is certainly not my first time Game Developing, but it was my first time making a 2D Sidescroller Game using Unity Game Engine. So, it was a new experience for me.
