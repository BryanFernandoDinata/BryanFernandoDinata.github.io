BUNNY LIFE

About This Project


This project is one of my oldest game Project. I did this project when I was learning about Game Development using Unity Game Engine. This is a 2D Top Down Game where the player need to dodge obstacles, collect carrots, and talk with other NPCs in order to finish the game. The other reason why I did this project is because I'm bored.


Doing this project was really fun. This is certainly not my first time Game Developing, but it was my first time making a Simulator Game using Unity Game Engine. So, it was a new experience for me.
